Manual de Usuario

Nombre del proyecto: JUNCOS - Sistema carga de Archivos JUNCOS 2019

Página Inicio:

En la página de inicio se podrá visualizar imágenes alusivas a competiciones anteriores de JUNCOS en modo de un carrusel de imágenes. Además, también se podrá redireccionar y observar el manual o guia de usuario en el cual se explicara más a fondo cómo funciona la plataforma y las distintas funcionalidades que se pueden realizar sobre la misma.

Iniciar Sesión:

Para iniciar sesión es necesario presionar el link “botón” de la esquina superior derecha llamado “Login”.

Una vez presionado el botón nos saldrá el cuadro de Login donde debemos colocar nuestro correo y contraseña.

Una vez que se ingresen los datos, se presiona el botón de “Login” el cual se muestra en color azul. Si los datos son incorrectos mostrará el respectivo error y le da la opción de volver iniciar sesión. En caso contrario mostrará el mensaje de Inicio secion con éxito y mostrará la página principal y habilitara todas las secciones de la página.

Página Inicio Manual:

En la página de inicio del manual de usuario se desplegará información relevante para los usuarios finales de la plataforma, en este caso los encargados (administradores) de darle soporte a la aplicación de JUNCOS 2019 y estar administrando los datos de la misma. En dicha sección o página, los administradores podrán observar y consultar una guia o manual de usuario en el cual se le explicara más a fondo al lector de cómo está estructurada la página, cómo funciona y qué acciones se pueden ejercer en la misma. Esta sección se puede acceder con o sin haber iniciado sesión.

Mostrar Información:

En la barra de navegación de la izquierda se muestran las diferentes secciones de la página, cada una muestra la respectiva tabla ya sea de participantes, lugares, universidades, medallero y eventos. Si le damos click sobre una de estas secciones nos mostrará la información almacenada en esa sección en la base de datos.

En la imagen anterior se puede observar cómo se visualizará la información relevante a cada colección o categoría. La misma será desplegada de la misma forma para todas las demás secciones, ajustándose al formato de cada una y sus respectivos campos.

Agregar información desde un CSV:

Para agregar información desde un csv es necesario irse a la sección donde se quiere subir la información, después debemos presionar el botón de “Importar”.

Una vez dado click en el botón de importar, este le presentara una tabla vacía que se llenara con los datos del CSV a importar y el botón de Choose File en el cual se podrá seleccionar el archivo CSV

que se quiere importar. Una vez dado en el botón de Choose File, este le abrirá una ventana donde podrá seleccionar el archivo tipo CSV que se quiere importar.

Finalmente, se cargará los datos del CSV en forma de tabla. Si desea subir dicha información a la base de datos únicamente se le tiene que dar en el botón de Subir.

Agregar información desde la tabla:

Una vez presionado el botón de “Agregar”, nos abre una ventana con todos los espacios necesarios de cada tabla. Una vez que se llenen los datos se presiona el botón “Aceptar” y si cumple con las validaciones se inserta de lo contrario mostrará el error respectivo. Se muestra el botón de agregar, en este caso se va agregar una universidad.

Se abre ventana donde se puede ingresar los datos respectivos a esa colección.

Se muestra los nuevos valores agregado y su mensaje respectivo.

Editar información desde la tabla:

Se carga una pestaña (Se modificara la fila de Alajuela)

Se abre sección donde se puede modificar los datos de la fila seleccionada

Se muestra la tabla con la fila editada y el mensaje respectivo.

Eliminar información desde la tabla:

Se muestra la tabla original antes de eliminar (Se eliminará la primera fila) Hacer click en la fila a eliminar, se muestra la siguiente ventana:

Finalmente se hace click en eliminar.

Se muestra la tabla luego de eliminar la fila y el mensaje respectivo.

Logout del Sistema:

Para terminar la sesión en el sitio únicamente será necesario presionar el botón de Logout en la esquina superior derecha de la plataforma.
